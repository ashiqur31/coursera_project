# Coursera Web Development Project
This repository contains a multi-page web application developed as part of a Coursera web development certification. It showcases fundamental front-end skills, including responsive layout design, interactive JavaScript elements, and asset management.

## 🚀 Features
- **Responsive Landing Page**: A clean entry point (index.html) optimized for different screen sizes.

- **Interactive UI**: Dynamic behavior powered by custom JavaScript.

- **Multi-page Navigation**: Dedicated sections for "About" and "Contact" information.

- **Live Deployment**: Hosted via GitHub Pages.

## 🛠️ Tech Stack
- **HTML5**: Structured content and semantic markup.

- **CSS3**: Custom styling and layout logic.

- **JavaScript**: Client-side logic and event handling.

## 📂 Project Structure
```
coursera_project/
├── about/                  # "About Us" page content and assets
├── contact/                # "Contact Us" page and form logic
├── images/                 # Project-wide image assets (logos, banners)
├── index.html              # Main application entry point
├── javascript_project.css  # Primary stylesheet
├── javascript_project.js   # Core JavaScript logic
└── README.md               # Project documentation
```

## ⚙️ Logic & Styling Details

### 1. JavaScript Logic (javascript_project.js)
The core script handles the "heavy lifting" for the user interface. Key functions include:

- **Interactive Triggers**: Listens for user clicks and scrolls to initiate UI changes.

- **Form Validation**: Ensures user input in the contact section meets specific criteria before submission.

- **Dynamic Content**: Updates the DOM to show or hide elements based on user interaction (e.g., mobile menu toggles).

### 2. Styling Architecture (javascript_project.css)
The project uses a custom CSS architecture designed for responsiveness:

- **Flexbox/Grid**: Used for modern, alignment-aware layouts.

- **Media Queries**: Ensures the site scales gracefully from mobile phones to desktop monitors.

- **Consistency**: Centralized variables for colors and fonts used across the about and contact subfolders.
